## TL;DR

Docling is IBM Research's open-source library for converting PDFs and other document formats into LLM-ready structured output. This video (part 4 of the series) introduces Docling as the preparation layer before feeding documents into an LLM pipeline — handling parsing, chunking, and metadata extraction so the model sees clean, structured content.

## Key Concepts

- Docling as the pre-processing stage in an LLM/RAG pipeline
- Converts PDFs, DOCX, PPTX, HTML into structured markdown or JSON
- Preserves document layout (tables, figures, headings) for better chunking
- Part 4 of the playlist — builds on earlier episodes covering format-specific parsing
- Designed to be LLM-agnostic — output feeds any downstream model or vector store
